React.createElement

  • jsx会被babel编译为React.createElement,所以React必须引入，否则编译后的代码将找不到React变量而报错

  • <App />代表这ReactElement，App代表React Element

React.createRef

ref的创建方式有三种

  1. 字符串的形式 已标记为不推荐使用

  2. ref={el=>this.divEle=el}

  3. React.createRef

    主要关注createRef

​ 函数式组件中不能直接使用ref 文档，可以在props中传递ref，或者使用forwardRef
