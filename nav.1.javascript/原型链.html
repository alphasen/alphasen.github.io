# 什么是原型

每一个 JavaScript 对象 (null 除外) 在创建的时候就会与之关联另一个对象，这个对象就是我们所说的原型，每一个对象都会从原型 "继承" 属性

# es5继承和es6继承的区别

  1. es6中先创建父类this，再用子类对其进行修饰（super关键字的特性）；而es5则是先创建子类实例，再用父类构造函数进行修饰。
  2. es5子类构造函数的__proto__指向Function.prototype，而es6子类构造函数的__proto__则是指向其父类的prototype
     class P6 {}

 class C6 extends P6{}

 console.log(C6.__proto__ === P6,C6.__proto__.__proto__ === P6.__proto__,C6.__proto__.__proto__ === Function.prototype) // true true true

     function P5(){}
 function C5(){
     P5.call(this)
 }
 C5.prototype= Object.create(P5.prototype)
 C5.prototype.constructor = C5

 console.log(C5.__proto__ === P5.__proto__, C5.__proto__ === Function.prototype) // true true
  3. class的特性
    1. 有暂时性死区TDZ
    2. 声明内部自动使用严格模式
    3. class内部所有的方法都不可枚举
    4. class内部所有方法都没有[[constructor]]
    5. class的调用必须使用new关键字
    6. class内部不能重写类名
