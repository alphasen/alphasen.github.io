# JavaScript代码是如何执行的

JavaScript是脚本语言，是解释执行的，不同于一般脚本语言，JavaScript是一段一段的执行的，在执行前会进行短时间的准备工作：变量提升，构建上下文等

# JavaScript如何管理上下文

JavaScript引擎内部有一个上下文栈，栈底是JavaScript的全局上下文，提供JavaScript核心能力，在执行函数时会构建函数的执行上下文并将其入栈，函数执行完后会将该上下文出栈；栈空间有限，超长函数调用链会栈溢出从而导致程序崩溃
